# 📱 微信小游戏部署完整教程

## 第一步：注册微信小程序账号（必需）

### 1.1 访问注册页面
```
https://mp.weixin.qq.com/
点击右上角 "立即注册"
```

### 1.2 选择账号类型
```
选择：小程序
（不要选错：订阅号/服务号/企业号/小程序）
```

### 1.3 填写账号信息
```
邮箱：未注册过微信公众平台的邮箱
密码：设置登录密码
```

### 1.4 邮箱激活
```
1. 收取激活邮件
2. 点击链接激活
3. 完成注册流程
```

### 1.5 选择主体类型

#### 个人类型（推荐新手）
```
优点：
- 免认证费（¥0）
- 注册快速（10分钟）
- 无需企业资质

缺点：
- 无法开通支付功能
- 无法认证（某些功能受限）
- 个人信息会显示

所需材料：
- 身份证正反面照片
- 管理员微信扫码
```

#### 企业类型
```
优点：
- 功能完整
- 可开通支付
- 可认证（¥300/年）

缺点：
- 需要企业资质
- 认证需要费用

所需材料：
- 营业执照
- 对公账户/法人微信
- 认证费 ¥300
```

### 1.6 完成注册
```
1. 提交主体信息
2. 管理员扫码验证
3. 等待审核（通常即时通过）
4. 注册完成！
```

---

## 第二步：配置小程序信息

### 2.1 登录小程序后台
```
https://mp.weixin.qq.com/
用注册的邮箱和密码登录
```

### 2.2 完善小程序信息
```
进入：设置 → 基本设置

需要填写：
✓ 小程序名称：种树消消乐
✓ 小程序简介：边玩游戏边种树，打造你的虚拟森林！
✓ 小程序头像：上传游戏图标（144x144）
✓ 服务类目：游戏 - 休闲游戏
```

### 2.3 获取 AppID
```
位置：设置 → 开发设置 → 开发者ID

AppID 示例：wx1234567890abcdef

⚠️ 重要：复制保存这个 AppID，后面会用到！
```

---

## 第三步：下载并安装微信开发者工具

### 3.1 下载工具
```
访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

选择你的系统：
- macOS
- Windows 64
- Windows 32
```

### 3.2 安装
```
macOS: 打开 .dmg 文件，拖拽到 Applications
Windows: 运行 .exe 安装程序
```

### 3.3 启动并登录
```
1. 打开微信开发者工具
2. 使用微信扫码登录
3. 选择小程序项目
```
