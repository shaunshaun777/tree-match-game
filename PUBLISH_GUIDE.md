## 🚀 详细步骤

### 第一步：注册微信小程序账号

1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册"
3. 选择"小程序" 类型
4. 填写注册信息
5. 激活邮箱
6. 进行主体认证（需要300元认证费）

**认证材料**:
- 企业：营业执照、法人身份证
- 个人：身份证（个人类型功能受限）

### 第二步：申请小游戏类目

1. 登录小程序后台
2. 设置 → 基本设置 → 服务类目
3. 添加类目："游戏"
4. 提交资质证明（如有）
5. 等待审核通过

### 第三步：获取AppID

1. 小程序后台 → 设置 → 开发设置
2. 复制 AppID
3. 在微信开发者工具中填写此AppID

### 第四步：配置服务器域名（如需联网）

1. 设置 → 开发设置 → 服务器域名
2. 添加你的域名：
   - request合法域名
   - uploadFile合法域名
   - downloadFile合法域名
   - udp合法域名

**域名要求**:
- 必须是 HTTPS
- 必须有ICP备案
- 一个月最多修改5次

### 第五步：开通流量主（广告）

**开通条件**:
- 小游戏累计独立访问用户（UV）不低于1000
- 无严重违规记录

**开通步骤**:
1. 推广 → 流量主 → 开通
2. 阅读并同意协议
3. 填写结算信息
4. 创建广告位
5. 获取广告位ID

**广告位类型**:
- 激励式广告（推荐）
- Banner广告
- 插屏广告
- 格子广告

### 第六步：配置代码

#### 1. 修改 game.json
```json
{
  "deviceOrientation": "portrait",
  "showStatusBar": false,
  "networkTimeout": {
    "request": 10000
  }
}
```

#### 2. 修改广告位ID (js/ad-manager.js)
```javascript
this.adConfig = {
  rewardedVideoAdUnitId: 'adunit-YOUR_REAL_ID_HERE',
  bannerAdUnitId: 'adunit-YOUR_REAL_ID_HERE',
  interstitialAdUnitId: 'adunit-YOUR_REAL_ID_HERE'
};
```

#### 3. 替换临时emoji为真实图片
```javascript
// 将 emoji 替换为图片路径
elements: [
  { name: 'seed', image: 'assets/images/elements/seed.png' },
  { name: 'water', image: 'assets/images/elements/water.png' },
  // ...
]
```

### 第七步：本地测试

#### 在微信开发者工具中测试
```bash
1. 打开微信开发者工具
2. 导入项目（选择 tree-match-game 文件夹）
3. 填写 AppID
4. 点击"编译"
5. 测试所有功能
```

#### 真机调试
```bash
1. 点击"真机调试"
2. 扫码在手机上打开
3. 测试触摸操作
4. 检查性能和兼容性
```

### 第八步：体验版测试

```bash
1. 点击"上传"
2. 填写版本号和描述
3. 上传成功后，在小程序后台可以看到"开发版本"
4. 生成体验版二维码
5. 分享给测试人员
6. 收集反馈并修复BUG
```

### 第九步：提交审核

#### 1. 填写基本信息
```
小程序后台 → 版本管理 → 开发版本 → 提交审核

必填信息：
- 小游戏名称
- 简介（< 120字）
- 游戏标签（最多5个）
- 游戏类型
- 是否含有付费内容
- 截图（至少3张，最多5张）
```

#### 2. 填写审核说明
```
测试账号（如有登录）：
- 账号：test123
- 密码：123456

游戏说明：
这是一款消除类益智游戏，玩家通过消除相同元素获得分数...

特殊功能说明：
- 包含激励视频广告
- 包含分享功能
- 无付费内容
```

#### 3. 审核注意事项
- ✅ 确保游戏可以正常运行
- ✅ 没有诱导分享
- ✅ 广告位置合理（不能强制观看）
- ✅ 内容健康，无违规信息
- ✅ 用户隐私保护（隐私政策）
- ❌ 不要在审核期间修改代码

### 第十步：等待审核

**审核时长**: 通常1-7个工作日

**审核状态**:
- 审核中
- 审核通过
- 审核失败（会说明原因）

**常见审核失败原因**:
1. 游戏无法正常运行
2. 诱导分享
3. 强制观看广告
4. 内容违规
5. 缺少必要的用户协议/隐私政策

**如果审核失败**:
1. 查看失败原因
2. 修改代码
3. 重新提交审核

### 第十一步：发布上线

```bash
审核通过后：

1. 小程序后台 → 版本管理 → 审核版本
2. 点击"发布"
3. 确认发布
4. 等待几分钟，小游戏正式上线
5. 生成小程序码/二维码进行推广
```

## 💰 收入相关

### 广告收入

**计费方式**:
- eCPM（千次展示成本）: 15-50元
- 实际收入 = 广告展示次数 / 1000 × eCPM × 分成比例

**分成比例**:
- 开发者分成：30-50%
- 微信平台：50-70%

**提现**:
- 每月1-10号结算上月收入
- 满100元可提现
- T+7天到账

### 内购收入（如有）

**分成比例**:
- iOS：苹果30% + 微信30% = 开发者40%
- Android：微信30% = 开发者70%

## ⚠️ 注意事项

### 合规要求
1. ✅ 必须有用户协议和隐私政策

1. ✅ 包体大小 < 4MB

3. ✅ FPS > 30
4. ✅ 内存占用 < 200MB
5. ✅ 适配所有主流机型
