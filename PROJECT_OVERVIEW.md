## 🚀 快速开始

### 1️⃣ 环境准备
```bash
# 下载并安装微信开发者工具
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

# 注册微信小程序账号
https://mp.weixin.qq.com/
```

### 2️⃣ 本地测试
```bash
# 方法1: 使用微信开发者工具（推荐）
1. 打开微信开发者工具
2. 导入项目（选择 tree-match-game 文件夹）
3. 填写测试 AppID（或使用测试号）
4. 点击"编译"运行

# 方法2: 浏览器测试（无广告功能）
cd tree-match-game
open pages/game/game.html
# 或使用 HTTP 服务器
python3 -m http.server 8000
# 访问 http://localhost:8000/pages/game/game.html
```

### 3️⃣ 核心配置

**修改广告位ID** (`js/ad-manager.js`):
```javascript
this.adConfig = {
  rewardedVideoAdUnitId: 'adunit-YOUR_ID',  // 替换为真实ID
  bannerAdUnitId: 'adunit-YOUR_ID',
  interstitialAdUnitId: 'adunit-YOUR_ID'
};
```
