# 🎨 美术资源需求清单

## 📝 资源规范

### 通用要求
- **格式**: PNG（透明背景）、JPG、MP3
- **命名**: 小写英文+下划线，如 `tree_pine_stage_1.png`
- **优化**: 使用 TinyPNG 压缩，尽可能减小文件大小
- **分辨率**: 提供 @2x 版本（适配高分屏）
- **总大小限制**: 小游戏包体 < 4MB

## 🎵 音效

| 音效名称 | 时长 | 说明 | 大小限制 |
|---------|------|------|---------|
| match.mp3 | < 1秒 | 消除音效，清脆 | < 50KB |
| success.mp3 | 1-2秒 | 关卡完成，欢快 | < 100KB |
| fail.mp3 | 1-2秒 | 失败音效，低沉 | < 100KB |
| tree_grow.mp3 | 1-2秒 | 树木成长，魔法音 | < 100KB |
| water.mp3 | < 1秒 | 浇水音效 | < 50KB |
| button_click.mp3 | < 0.5秒 | 按钮点击 | < 30KB |
| bgm_game.mp3 | 循环 | 游戏背景音乐，轻快 | < 500KB |
| bgm_forest.mp3 | 循环 | 森林氛围音乐，舒缓 | < 500KB |

**音效要求**:
- 格式: MP3
- 采样率: 44.1kHz
- 比特率: 128kbps（背景音乐）/ 64kbps（音效）
- 声道: 单声道（音效）/ 双声道（背景音乐）
